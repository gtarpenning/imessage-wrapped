# Agent Instructions

## Project: iMessage Wrapped
Python CLI tool to export/analyze iMessage data from macOS SQLite DB.

## Commands
- Format/lint: `ruff check src/` and `ruff format src/`
- Type check: `ty check`
- Run tool: `mexport` (entry point in cli.py)

## Testing
No formal tests. Verify by running `mexport analyze` and checking terminal output displays correctly.
